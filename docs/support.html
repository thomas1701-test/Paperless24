Häufige Fragen

Die App verbindet sich nicht mit meinem Server.
Stelle sicher, dass die Server-URL korrekt eingegeben ist (inkl. Protokoll, z.B. http://192.168.1.10:8000). Der Server muss vom iPhone aus erreichbar sein. Bei HTTPS-Servern muss das Zertifikat gültig sein.

Login funktioniert nicht.
Überprüfe Benutzername und Passwort in deinen Paperless-ngx-Einstellungen. Ein API-Token kann alternativ in den App-Einstellungen hinterlegt werden.

Scan-Funktion startet nicht.
Die App benötigt Kamerazugriff. Gehe zu Einstellungen → Paperless TeDi → Kamera und erlaube den Zugriff.

Face ID fragt immer wieder nach dem Login.
Stelle sicher, dass Face ID in den App-Einstellungen aktiviert ist und die App Berechtigung für Face ID hat (Einstellungen → Paperless TeDi → Face ID).

Voraussetzungen

  • iOS 17 oder neuer
  • Eigener Paperless-ngx-Server (Version 2.0+)
  • Netzwerkverbindung zum Server (lokal oder per VPN/Domain)
