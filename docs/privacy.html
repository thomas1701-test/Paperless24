Kurzfassung: Paperless TeDi sammelt keine persönlichen Daten und sendet keine Daten an Dritte. Alle Daten verbleiben auf deinem eigenen Server.

1. Verantwortlicher

Thomas Dillmann
E-Mail: thomas.dillmann@gmx.net

2. Welche Daten werden verarbeitet?

Paperless TeDi verbindet sich ausschließlich mit dem Paperless-ngx-Server, den du selbst konfigurierst. Die App verarbeitet lokal:

  • Die von dir eingegebene Server-URL und Zugangsdaten (gespeichert im iOS Keychain)
  • Dokumente und Metadaten, die von deinem Server abgerufen werden
  • Kamerabilder beim Scannen (nur zur Übertragung an deinen Server, nicht gespeichert)

3. Datenübertragung an Dritte

Es findet keine Übertragung von Daten an Dritte statt. Die App kommuniziert ausschließlich mit dem Server, den du selbst betreibst. Es gibt keine Analyse-Dienste, keine Werbung und kein Tracking.

4. Lokale Netzwerkverbindung

Die App nutzt das lokale Netzwerk, um deinen Paperless-ngx-Server zu erreichen. iOS fragt einmalig um Erlaubnis für den lokalen Netzwerkzugriff. Diese Verbindung wird ausschließlich für die Kommunikation mit deinem eigenen Server verwendet.

5. Kamera & Fotos

Der Kamerazugriff wird nur zum Scannen von Dokumenten genutzt. Aufgenommene Bilder werden direkt an deinen Paperless-ngx-Server übertragen und nicht dauerhaft auf dem Gerät gespeichert.

Der Fotobibliothek-Zugriff wird nur verwendet, wenn du explizit ein Foto aus der Bibliothek importierst.

6. Face ID / Biometrie

Face ID wird ausschließlich zur lokalen Entsperrung der App verwendet. Biometrische Daten verlassen das Gerät nicht und werden nicht gespeichert oder übertragen.

7. Datensicherheit

Zugangsdaten (Server-URL, Benutzername, API-Token) werden ausschließlich im iOS Keychain gespeichert, der durch das Betriebssystem verschlüsselt und geschützt wird.
